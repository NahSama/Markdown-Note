-   File is grp of information stored in hard-disk. It can occupy 1 or more blocks of hard-disk
-   File attributes (location, ..) is kept in **directory-structure**

# File attributes

-   name
-   id
-   type
-   location (of content in the hard disk)
-   size
-   protection
-   last used

# File operations

-   creation
-   read/write
-   reposition
-   delete
-   truncating (delete content only, not the whole file)

# Directory structure

can be _tree_ or _acyclic graph_ in case of needing to share the files between different users / directory
